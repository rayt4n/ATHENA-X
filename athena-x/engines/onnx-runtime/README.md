# ONNX Runtime (Browser)

Serves browser-runnable ONNX models (Change 4):
LightGBM-small, Random Forest, Logistic Regression.
Maintains model registry, downloads .onnx files from Supabase Storage,
caches in IndexedDB, runs inference via onnxruntime-web (WebGPU).
