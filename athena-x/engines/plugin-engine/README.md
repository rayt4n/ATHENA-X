# Plugin Loader & Lifecycle

Discovers, loads, and manages plugin lifecycle (Change 13).
Validates plugin manifests against schemas/plugins/manifest.schema.json.
Maintains a plugin registry. Hot-reloads plugins in development.
Used by all agents that need indicator/options/pattern computations.
