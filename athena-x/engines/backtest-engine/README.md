# Backtest Engine (vectorbt)

Real Python backtesting engine (Change 6 of STEP 2.1 — never mock).
Uses vectorbt to run strategies over historical data.
Computes metrics: returns, Sharpe, Sortino, MaxDD, Calmar, win rate.
Strategies: athena-ensemble, ta-only, news-only, options-only, macro-only, forecast-only.
